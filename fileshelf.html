⚙️ One-time Setup

Open this HTML file in a text editor and paste your Firebase config at the top. Here's how to get it:

1. Go to console.firebase.google.com and sign in with Google
2. Click "Add project" → name it anything → click through the steps → Create project
3. Inside your project, click "Web" (</>) to register a web app → give it any name → click Register
4. You'll see a block of code with your config — copy the 6 values (apiKey, authDomain, etc.) and paste them at the top of this HTML file where it says PASTE_YOUR_..._HERE
5. In Firebase, go to Build → Storage → Get started → click through → then go to the Rules tab and paste this rule:

allow read, write: if true;
Click Publish
6. Save and reload this HTML file. Done ✅
💡 This app works from any browser — phone, laptop, anywhere. Host it for free on Netlify (drag & drop the file) or just open it directly from your device.
